# The Gilded Bazaar

## 00.2 – Design Language & Art Direction

**Version:** 1.0
**Status:** Living Document

---

# Purpose

This document defines the complete visual identity of The Gilded Bazaar.

Every application screen, website, marketing asset, AI-generated illustration and animation should follow these standards.

The objective is to create a product that immediately feels recognisable, premium and timeless.

---

# Design Vision

The Gilded Bazaar should feel like opening an exquisitely illustrated fantasy tome.

Not a game.

Not a productivity app.

Not a virtual tabletop.

Instead, it should feel like a beautifully crafted companion book that quietly comes to life.

The design language combines:

* Luxury publishing
* Fantasy manuscripts
* Premium mobile applications
* Elegant board games
* Subtle magical realism

The result should be immersive without becoming cluttered.

---

# Emotional Goals

Users should experience:

* Wonder
* Warmth
* Curiosity
* Discovery
* Calm confidence

The interface should never feel stressful or noisy.

---

# Design Principles

## Elegant Before Ornate

Avoid excessive decoration.

Decorative elements should support usability rather than compete with it.

---

## Readability First

Fantasy should never reduce accessibility.

Text should remain clear under all lighting conditions.

Contrast should exceed accessibility guidelines.

---

## Premium Simplicity

Every screen should feel handcrafted.

Spacing should be generous.

Animations should be subtle.

Components should have room to breathe.

---

## Consistency Over Novelty

The same interaction should always behave the same way.

Users should learn the interface once.

# Typography

## Primary Typeface

Modern sans-serif.

Highly readable.

Used for:

* Body text
* Navigation
* Buttons
* Lists

---

## Secondary Typeface

Elegant serif.

Used sparingly for:

* Merchant names
* Settlement titles
* Quest headings
* Chapter dividers
* Certificates

Never use decorative fantasy fonts for body copy.

---

# Iconography

Icons should appear hand-crafted but modern.

Preferred style:

* Thin to medium stroke
* Rounded corners
* Slightly organic geometry

Avoid:

* Heavy gradients
* Cartoon icons
* Glossy effects
* Overly realistic artwork

---

# Cards

Every major feature is presented as a card.

Cards should feature:

* Soft rounded corners
* Subtle elevation
* Fine gold borders where appropriate
* Minimal shadows
* Clear hierarchy

Cards should never appear cluttered.

---

# Texture

Texture should be subtle.

Preferred:

* Fine parchment grain
* Light paper fibres
* Brushed metal
* Worn leather
* Ink wash

Avoid:

* Heavy noise
* Obvious repeating patterns
* Artificial grunge

---

# Lighting

Lighting should suggest candlelight.

Soft highlights.

Warm reflections.

Gentle glows around magical content.

Avoid harsh white lighting.

---

# Animation Philosophy

Animations communicate quality.

They should feel:

* Smooth
* Deliberate
* Magical
* Lightweight

Never distracting.

---

## Recommended Durations

Micro interactions

100–150 ms

Card transitions

200–250 ms

Page transitions

250–350 ms

Modal presentation

300 ms

Celebration animations

500–700 ms

---

# Magical Effects

Magic should be understated.

Examples:

* Floating particles
* Arcane glyphs
* Soft golden sparkles
* Gentle blue shimmer
* Candle embers

Never use excessive particle systems.

---

# Illustrations

Illustrations should resemble premium fantasy book artwork.

Influences include:

* Watercolour paintings
* Ink illustrations
* Hand-painted concept art
* Storybook lighting

Avoid:

* Hyper-realism
* Cartoon proportions
* Comic-book exaggeration

---

# Merchant Portraits

Every merchant should feel unique.

Portrait guidelines:

Half-body composition.

Interesting clothing.

Visible profession.

Expressive faces.

Warm lighting.

Consistent painterly style.

Backgrounds should reinforce personality without distracting from the subject.

---

# Item Artwork

Items should appear like museum catalogue illustrations.

Single subject.

Clean silhouette.

Minimal background.

Consistent lighting.

Premium presentation.

---

# World Maps

Maps should resemble hand-drawn atlases.

Muted colours.

Fine linework.

Readable labels.

Subtle relief shading.

Never imitate satellite imagery.

---

# Sound Design

Audio is optional.

When enabled:

* Soft page turns
* Coin clinks
* Quill writing
* Fireplace ambience
* Tavern murmurs
* Gentle magical chimes

Avoid repetitive or intrusive sounds.

---

# Marketing Style

Marketing should emphasise imagination over mechanics.

Screenshots should feel cinematic.

Avoid cluttering promotional images with excessive UI.

Focus on atmosphere, storytelling and craftsmanship.

---

# AI Artwork Standards

All AI-generated assets must follow the established art direction.

Prompts should consistently specify:

* Painterly fantasy
* Premium storybook quality
* Warm cinematic lighting
* Rich textures
* Elegant composition
* Consistent colour palette

Generated assets should be reviewed before inclusion.

---

# Accessibility

Beauty must never compromise usability.

Maintain:

* High contrast
* Large touch targets
* Clear typography
* Colour-independent status indicators
* Screen-reader compatibility

Accessibility is a core design requirement.

---

# Definition of Success

A user should recognise a screenshot of The Gilded Bazaar without seeing the logo.

The application should feel cohesive, elegant and unmistakably its own.

---
